Easily embed tappi reviews widget on a WordPress website

== Description ==

tappi reviews allows for easy embedding of reviews from [tappi](https://tappi.app) on a WordPress website.
Plugin use is subject to tappi's [terms](https://tappi.app/terms) and [privacy policy](https://tappi.app/privacy-policy).

== Installation ==

1. Login to your WordPress admin
1. Click on the Plugins tab
1. Click on the “Add New” button
1. Search for “tappi reviews”
1. Look for our plugin, click on “Install Now” and then on “Activate”
1. Once the plugin is installed and active, go to Settings > tappi reviews
1. Enter your account's join code in the join code settings field and save
1. If you want to allow reviews submission, you will need to add your Reviews Page URL
1. To display the reviews on any page or widget, enter the shortcode [tappi-reviews]
1. To display review submission form, enter the shortcode [tappi-reviews display_form=true]

= Alternative installation method =
1. Download and save the attached ZIP file
1. Once downloaded, installation works just as for other WordPress plugins
1. Activate the plugin then go to Settings > tappi reviews
1. Enter your account's join code in the join code settings field and save
1. If you want to allow reviews submission, you will need to add your Reviews Page URL
1. To display the reviews on any page or widget, enter the shortcode [tappi-reviews]
1. To display review submission form, enter the shortcode [tappi-reviews display_form=true]
